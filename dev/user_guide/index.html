User Guide

This guide covers practical workflows for using SymbolicDiagonalization.jl.

Basic Usage

Getting Eigenvalues

using SymbolicDiagonalization, Symbolics, LinearAlgebra
using Main: LaTeX

# Define symbolic variables
@variables a b c

# Create a matrix
M = [a b; b c]

# Get eigenvalues
vals = eigvals(M)

Matrix:

LaTeX(M)
\[ \left[ \begin{array}{cc} a & b \\ b & c \\ \end{array} \right] \]

Eigenvalues:

LaTeX(vals)

Eigenvalues:

\[ \begin{aligned} \lambda_{1} &= \frac{a - c - \sqrt{\left( a - c \right)^{2} + 4 ~ b^{2}}}{2} + c \\[0.3em] \lambda_{2} &= \frac{a - c + \sqrt{\left( a - c \right)^{2} + 4 ~ b^{2}}}{2} + c \end{aligned} \]

Getting Eigenvectors

@variables x y

# Symmetric matrix
S = [x y; y x]
E = eigen(S)

Eigenvectors (columns of E.vectors):

LaTeX(E.vectors)
\[ \left[ \begin{array}{cc} 1 & 1 \\ 1 & -1 \\ \end{array} \right] \]

Full Diagonalization

For M = P * D * P⁻¹ decomposition:

P, D, pairs = symbolic_diagonalize(S)

Diagonal matrix D:

LaTeX(D)
\[ \left[ \begin{array}{cc} x + y & 0 \\ 0 & x - y \\ \end{array} \right] \]

Working with Larger Matrices

Block-Diagonal Matrices

The package automatically detects block structure:

@variables a b c d e f

# 4×4 block-diagonal
B = [a  b  0  0;
     b  a  0  0;
     0  0  c  d;
     0  0  d  c]

vals = eigvals(B)

Matrix:

LaTeX(B)
\[ \left[ \begin{array}{cccc} a & b & 0 & 0 \\ b & a & 0 & 0 \\ 0 & 0 & c & d \\ 0 & 0 & d & c \\ \end{array} \right] \]

Eigenvalues (solved block-by-block):

LaTeX(vals)

Eigenvalues:

\[ \begin{aligned} \lambda_{1} &= a + b \\[0.3em] \lambda_{2} &= a - b \\[0.3em] \lambda_{3} &= c + d \\[0.3em] \lambda_{4} &= c - d \end{aligned} \]

Circulant Matrices

Any n×n circulant matrix has closed-form eigenvalues via the DFT:

@variables a b c d

# 4×4 circulant
C = [a b c d;
     d a b c;
     c d a b;
     b c d a]

vals = eigvals(C)

Matrix:

LaTeX(C)
\[ \left[ \begin{array}{cccc} a & b & c & d \\ d & a & b & c \\ c & d & a & b \\ b & c & d & a \\ \end{array} \right] \]

Eigenvalues:

LaTeX(vals)

Eigenvalues:

\[ \begin{aligned} \lambda_{1} &= a + b + c + d \\[0.3em] \lambda_{2} &= a - c + \left( b - d \right) ~ \mathit{i} \\[0.3em] \lambda_{3} &= a - b + c - d \\[0.3em] \lambda_{4} &= a - c + \left( - b + d \right) ~ \mathit{i} \end{aligned} \]

Kronecker Products

Eigenvalues of A ⊗ B are products of individual eigenvalues. A 4×4 Kronecker product that would require solving a quartic is reduced to two quadratics:

@variables a b c d

A = [a b; b a]
B = [c d; d c]
K = kron(A, B)

vals = eigvals(K)

The 4×4 Kronecker product:

LaTeX(K)
\[ \left[ \begin{array}{cccc} a ~ c & a ~ d & b ~ c & b ~ d \\ a ~ d & a ~ c & b ~ d & b ~ c \\ b ~ c & b ~ d & a ~ c & a ~ d \\ b ~ d & b ~ c & a ~ d & a ~ c \\ \end{array} \right] \]

Eigenvalues (products of 2×2 eigenvalues):

LaTeX(vals)

Eigenvalues:

\[ \begin{aligned} \lambda_{1} &= \left( a + b \right) ~ \left( c + d \right) \\[0.3em] \lambda_{2} &= \left( a + b \right) ~ c - \left( a + b \right) ~ d + 1.2246467991473532e-16 ~ \left( a + b \right) ~ d ~ \mathit{i} \\[0.3em] \lambda_{3} &= a ~ \left( c + d \right) - b ~ \left( c + d \right) + 1.2246467991473532e-16 ~ b ~ \left( c + d \right) ~ \mathit{i} \\[0.3em] \lambda_{4} &= a ~ c - a ~ d - b ~ c + b ~ d + \left( 1.2246467991473532e-16 ~ a ~ d + 1.2246467991473532e-16 ~ b ~ c - 2.4492935982947064e-16 ~ b ~ d \right) ~ \mathit{i} \end{aligned} \]

Rotation Matrices

SO(2): 2D Rotations

@variables θ

R = SO2_rotation(θ)
vals = eigvals(R)

Rotation matrix:

LaTeX(R)
\[ \left[ \begin{array}{cc} \cos\left( \theta \right) & - \sin\left( \theta \right) \\ \sin\left( \theta \right) & \cos\left( \theta \right) \\ \end{array} \right] \]

Eigenvalues e^{±iθ}:

LaTeX(vals)

Eigenvalues:

\[ \begin{aligned} \lambda_{1} &= \cos\left( \theta \right) + \sin\left( \theta \right) ~ \mathit{i} \\[0.3em] \lambda_{2} &= \cos\left( \theta \right) - \sin\left( \theta \right) ~ \mathit{i} \end{aligned} \]

SO(3): 3D Rotations

Rz = SO3_Rz(θ)
vals = eigvals(Rz)

Rotation about z-axis:

LaTeX(Rz)
\[ \left[ \begin{array}{ccc} \cos\left( \theta \right) & - \sin\left( \theta \right) & 0 \\ \sin\left( \theta \right) & \cos\left( \theta \right) & 0 \\ 0 & 0 & 1 \\ \end{array} \right] \]

Eigenvalues {1, e^{±iθ}}:

LaTeX(vals)

Eigenvalues:

\[ \begin{aligned} \lambda_{1} &= \cos\left( \theta \right) + \sin\left( \theta \right) ~ \mathit{i} \\[0.3em] \lambda_{2} &= \cos\left( \theta \right) - \sin\left( \theta \right) ~ \mathit{i} \\[0.3em] \lambda_{3} &= 1 \end{aligned} \]

Kronecker Products of Rotations

@variables θ φ

K = kron(SO2_rotation(θ), SO2_rotation(φ))
vals = eigvals(K)

Eigenvalues e^{i(±θ±φ)}:

LaTeX(vals)

Eigenvalues:

\[ \begin{aligned} \lambda_{1} &= \cos\left( - \theta - \varphi \right) + \sin\left( - \theta - \varphi \right) ~ \mathit{i} \\[0.3em] \lambda_{2} &= \cos\left( \theta - \varphi \right) + \sin\left( \theta - \varphi \right) ~ \mathit{i} \\[0.3em] \lambda_{3} &= \cos\left( - \theta + \varphi \right) + \sin\left( - \theta + \varphi \right) ~ \mathit{i} \\[0.3em] \lambda_{4} &= \cos\left( \theta + \varphi \right) + \sin\left( \theta + \varphi \right) ~ \mathit{i} \end{aligned} \]

Tips and Best Practices

1. Check for Structure First

Before calling eigvals(), consider if your matrix has exploitable structure:

@variables a b c d e f

# Good: Circulant structure detected automatically
C = [a b c; c a b; b c a]
eigvals(C)  # Instant via DFT

# Slower: Generic 3×3 symmetric (no special structure)
M = [a b c; b d e; c e f]
eigvals(M)  # Uses Cardano formula (larger expressions)

2. Use Constructors for Lie Groups

The built-in constructors ensure proper structure detection:

# Good: Uses SO2_rotation constructor
R = SO2_rotation(θ)

# Works but less clear: Manual construction
R = [cos(θ) -sin(θ); sin(θ) cos(θ)]

3. Handle Large Expressions

For complex matrices, expressions can grow large:

# Set timeout for expensive computations
eigvals(M; timeout=60)  # 60 seconds max

# Limit expression complexity
eigvals(M; max_terms=1000)

4. Numeric Verification

Verify symbolic results numerically:

@variables a b
M = [a b; b a]
vals = eigvals(M)

# Substitute numeric values
using Symbolics: substitute
val_numeric = substitute(vals[1], Dict(a => 1.0, b => 2.0))
# Should match: eigvals([1.0 2.0; 2.0 1.0])

Common Patterns

Symmetric Toeplitz Tridiagonal

Matrices with constant diagonals have closed-form eigenvalues:

@variables a b

# 4×4 symmetric Toeplitz tridiagonal
T = [a b 0 0;
     b a b 0;
     0 b a b;
     0 0 b a]

vals = eigvals(T)

Matrix:

LaTeX(T)
\[ \left[ \begin{array}{cccc} a & b & 0 & 0 \\ b & a & b & 0 \\ 0 & b & a & b \\ 0 & 0 & b & a \\ \end{array} \right] \]

Eigenvalues λₖ = a + 2b·cos(kπ/(n+1)):

LaTeX(vals)

Eigenvalues:

\[ \begin{aligned} \lambda_{1} &= a + 1.618 ~ b \\[0.3em] \lambda_{2} &= a + 0.61803 ~ b \\[0.3em] \lambda_{3} &= a - 0.61803 ~ b \\[0.3em] \lambda_{4} &= a - 1.618 ~ b \end{aligned} \]
