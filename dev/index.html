SymbolicDiagonalization.jl

Symbolic matrix diagonalization for Julia.

The Problem

The Abel-Ruffini theorem proves no closed-form solution exists for polynomials of degree 5+. General symbolic 5×5+ matrices cannot be diagonalized. However, matrices with exploitable structure can be solved regardless of size.

Usage

using SymbolicDiagonalization, Symbolics, LinearAlgebra
using Main: LaTeX

@variables a b c d
M = [a b 0 0; b a 0 0; 0 0 c d; 0 0 d c]

Block-diagonal matrix:

LaTeX(M)
\[ \left[ \begin{array}{cccc} a & b & 0 & 0 \\ b & a & 0 & 0 \\ 0 & 0 & c & d \\ 0 & 0 & d & c \\ \end{array} \right] \]

Eigenvalues (each block solved independently):

LaTeX(eigvals(M))

Eigenvalues:

\[ \begin{aligned} \lambda_{1} &= a + b \\[0.3em] \lambda_{2} &= a - b \\[0.3em] \lambda_{3} &= c + d \\[0.3em] \lambda_{4} &= c - d \end{aligned} \]

Examples

Circulant (any size)

@variables a b c d
C = [a b c d; d a b c; c d a b; b c d a]
LaTeX(C)
\[ \left[ \begin{array}{cccc} a & b & c & d \\ d & a & b & c \\ c & d & a & b \\ b & c & d & a \\ \end{array} \right] \]

Eigenvalues via DFT:

LaTeX(eigvals(C))

Eigenvalues:

\[ \begin{aligned} \lambda_{1} &= a + b + c + d \\[0.3em] \lambda_{2} &= a - c + \left( b - d \right) ~ \mathit{i} \\[0.3em] \lambda_{3} &= a - b + c - d \\[0.3em] \lambda_{4} &= a - c + \left( - b + d \right) ~ \mathit{i} \end{aligned} \]

Kronecker Product

@variables θ φ
K = kron(SO2_rotation(θ), SO2_rotation(φ))

Eigenvalues e^{i(±θ±φ)}:

LaTeX(eigvals(K))

Eigenvalues:

\[ \begin{aligned} \lambda_{1} &= \cos\left( - \theta - \varphi \right) + \sin\left( - \theta - \varphi \right) ~ \mathit{i} \\[0.3em] \lambda_{2} &= \cos\left( \theta - \varphi \right) + \sin\left( \theta - \varphi \right) ~ \mathit{i} \\[0.3em] \lambda_{3} &= \cos\left( - \theta + \varphi \right) + \sin\left( - \theta + \varphi \right) ~ \mathit{i} \\[0.3em] \lambda_{4} &= \cos\left( \theta + \varphi \right) + \sin\left( \theta + \varphi \right) ~ \mathit{i} \end{aligned} \]

Nested Kronecker (1024×1024)

# 10-fold Kronecker: 1024×1024 matrix, 30 parameters
# Degree-1024 polynomial reduced to 10 quadratics
matrices = [[Symbolics.variable(Symbol("a$i")) Symbolics.variable(Symbol("b$i"));
             Symbolics.variable(Symbol("b$i")) Symbolics.variable(Symbol("c$i"))] for i in 1:10]
K = reduce(kron, matrices)
eigvals(K)  # ~33 seconds

Path Laplacian

L = path_laplacian(5)
LaTeX(L)
\[ \left[ \begin{array}{ccccc} 1 & -1 & 0 & 0 & 0 \\ -1 & 2 & -1 & 0 & 0 \\ 0 & -1 & 2 & -1 & 0 \\ 0 & 0 & -1 & 2 & -1 \\ 0 & 0 & 0 & -1 & 1 \\ \end{array} \right] \]

Eigenvalues λₖ = 2 - 2cos(πk/n):

LaTeX(eigvals(L))

Eigenvalues:

\[ \begin{aligned} \lambda_{1} &= 4.99692479230196e-17 \\[0.3em] \lambda_{2} &= 0.3819660112501052 \\[0.3em] \lambda_{3} &= 1.3819660112501049 \\[0.3em] \lambda_{4} &= 2.6180339887498953 \\[0.3em] \lambda_{5} &= 3.6180339887498945 \end{aligned} \]

API

Main Functions

eigvals(M)                    # Eigenvalues (LinearAlgebra interface)
eigen(M)                      # Eigen decomposition
symbolic_eigenvalues(M)       # Returns (values, poly, λ)
symbolic_eigenpairs(M)        # Returns (pairs, poly, λ)
symbolic_diagonalize(M)       # Returns (P, D, pairs) where M = P*D*P⁻¹

Constructors

# Rotations
SO2_rotation(θ)
SO3_Rx(θ), SO3_Ry(θ), SO3_Rz(θ)
SU2_Ux(θ), SU2_Uy(θ), SU2_Uz(θ)

# Kronecker
SO2_kron([θ, φ, ...])
SU2_kron([θ, φ, ...])

# Graphs
path_laplacian(n)
cycle_laplacian(n)

# Transforms
hadamard_matrix(n)  # 2ⁿ×2ⁿ
dft_matrix(n)

# Cartan
cartan_matrix_A(n)
cartan_matrix_G2()

Limitations

  • 5×5+ without structure: No solution exists (Abel-Ruffini)
  • Large expressions: 4×4 quartic formulas produce large expressions
